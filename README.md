# Создание своего виртуального компьютера
*Тут содержаться файлы из обучающего курса "Nand2Tetris" по книге "Архитектура компьютерных систем" Ноам Нисан и Шимон Шокен.*
### Состав проектов
1. В проекте 1 содержаться базовые логические модули начиная от простейшего вентиля *Nand*.
2. В проекте 2 содержаться более сложные чипы, состоящие из модулей проекта 1, а также финал этого проекта Апапратно-логическкое устройство (АЛУ).
